Este repositorio contiene dos implementaciones de grafos en Java como parte de los ejerciocios prácticos de la asignatura Estructura de Datos II

Ejercicio 1 : Grafo Dirigido y Ponderado usando Listas de Adyacencia
Ejercicio 2 : Grafo No Dirigido usando Matriz de Adyacencia

Ambos programas implementan los recorridos BFS(Búsqueda en Anchura) y DFS(Búsqueda en Profundidad)
